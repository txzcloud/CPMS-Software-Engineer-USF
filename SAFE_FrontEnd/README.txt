Install XAMPP: https://www.apachefriends.org/index.html
Install Microsoft SQL Server Configuration: https://www.microsoft.com/en-us/sql-server/sql-server-downloads
Install Microsoft SQL Server Management Studio(SSMS): https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15
Set up Environment to build and read PHP, HTML/CSS, and JS source code. 

Steps to execute project:
1. Download source code and save it inside the XAMPP->htdocs Folder.
2. Open and Build the project on Visual Studio or preferred environment.
3. Run Xampp on local host and Start Apache and Tomcat Server.
4. Run SSMS and login local machine.
5. In the source code, download the SQL Script and execute inside of SSMS to build database.

On a preferred web browser, direct to localhost/{path of source code downloaded}

NB: Ensure in SQL Server Configuration (Windows) that "MSSQL" and "SQL Agent" are both running.
NB: Ensure to change the server name, database's username, and password in the source code to match your local machine in order to connect successfully.
